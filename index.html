Rules of Game

Just like tic tac toe, this game is played between two players, adding a symbol to the board in turns. However, unlike tic tac toe, the players use the same symbol (a cross), and can both contribute to the same line. Interestingly, although this is a simplification of tic tac toe, it actually improves the game by making a draw impossible instead of practically guaranteed. Note that in the game below, each player's symbol is drawn in a same colour.

How to Play?

This is multiplayer Game. You have two options 'Create Game' and 'Join Game'. Before creating a game, you will need to input your name and select the rows, columns and length. The Rows and Column specfies the dimension of the grid. The length specifies the winning condition of the game e.g '3' length means that the game will end when a player makes a 3 in a row of 'X'. After pressing the 'Create Game' button, you will be provided with a Room ID. For joining the game, input your name first and ask your friend to share the Room ID with you.
